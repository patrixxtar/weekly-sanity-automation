About Transmission

How to add download movies:

  1. Login to the GUI (credentials sent on discord)
  2. Copy any torrent download links
  3. On Transmission GUI, click Add +
  4. Enter your movie download link, hit OK
  5. Once done, movie will be automatically added to Jellyfin

About Jellyfin

Jellyfin is a volunteer-built media solution that puts you in control of your entertainment. Stream movies, TV shows, and music directly from your server setup to any device without trackers.

How to add subtitles
For PC and Mobile
  1. Select the movie
  2. Click More options > Edit subtitles
  3. Choose your language then hit Search icon
  4. Look for "Perfect Match" tags on the result then tap Download
  5. Once downloaded, movie should have subtitles added
How to edit subtitle
For PC and Mobile
  1. Go to Settings by clicking the user icon
  2. Select Subtitles
  3. Look for "Subtitle Appearance"
  4. Modify values as needed
  5. Click Save

About n8n

n8n is an extendable workflow automation tool. It allows you to build custom node-based automations, connect different apps, and parse webhook or JSON payloads effortlessly.

About PiKaraoke

PiKaraoke turns your local environment into a full hosting platform for karaoke parties. Queue up your favorite tracks, manage songs from your smartphone, and start singing!

About OpenClaw

OpenClaw is an open-source modern engine implementation for Captain Claw (1997), recreating the classic platforming experience flawlessly natively on modern systems.

About Code-Server

Code-Server runs VS Code in the cloud. Write code, access a built-in terminal, and manage your workspace directly through any browser window.

Usage Instructions
  1. Click "Open Editor" and enter your workspace password.
  2. The environment points directly to your home files at /home/scarlett.
  3. You can edit configs or launch containers from the browser terminal.

About AdGuard Home

AdGuard Home acts as a private network ad and tracking domain filtering node. It blocks ad trackers at the network level before they load.

Usage Instructions
  1. Click "DNS Controls" to view the statistics block panel.
  2. Configure your remote client devices to look at this server IP for DNS resolution.
  3. Ensure your home IP is listed in "Allowed Clients" to secure backend routing.
